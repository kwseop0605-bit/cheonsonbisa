---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 무협 RPG + 생활/제작 시뮬레이션 |

## 2. 스토리 시나리오

### 배경
전설의 무공서 **천마신공(天魔神功)**이 사라진 지 백 년. 플레이어는 이름 없는 무명검사로, 우연히 노인에게서 낡은 두루마리를 건네받으면서 운명이 시작된다.

### 주요 등장인물

| 인물 | 역할 |
|------|------|
| 무명검사 | 플레이어 캐릭터 |
| 노인 (이름 불명) | 프롤로그에서 두루마리를 건네주고 사라지는 인물 |
| 대장장이 오씨 | 제련·제작법을 가르쳐주는 NPC |
| 약초꾼 이씨 | 회복단·내공단 제조법을 가르쳐주는 NPC |
| 원로 방씨 | 청룡방 정보를 제공하고 퀘스트를 의뢰하는 NPC |
| 혈귀 장옥 | 청룡방 두목, 최종 보스 |

### 챕터 흐름

```
[프롤로그] 주막
  → 노인이 두루마리(천마신공 第一式) 전달
  → 두루마리 수락 or 무시 (선택지)

[마을 어귀]
  → 청룡방 등장 소식
  → 자유 탐색: 대장장이 / 약초꾼 / 원로 / 뒷산

[대장장이 오씨]
  → 철검날·칼자루·칼집·갑옷 제작법 전수
  → 제련/제작 탭 해금

[약초꾼 이씨]
  → 회복단·내공단 제조법 전수

[원로 방씨]
  → 청룡방 두목 혈귀 장옥 정보 제공
  → 뒷산 토벌 퀘스트 의뢰

[뒷산 동굴 입구]
  → 졸개와 전투 or 잠입 (선택지)

[동굴 내부]
  → 졸개 격파 후 두목과 대면
  → 두루마리 사수 or 포기 (선택지)

[최종 전투] 혈귀 장옥
  → 승리: 천하무적 엔딩
  → 패배: 전패 엔딩
  → 두루마리 포기: 포기 엔딩
```

## 3. 전투 시스템

### 기본 구조
- 턴제 전투
- 플레이어는 보유한 무공 스킬 중 선택하여 행동
- 적은 자동으로 반격

### 스탯

| 스탯 | 설명 |
|------|------|
| 체력 (HP) | 0이 되면 패배 |
| 내공 (MP) | 스킬 사용 시 소모 |
| 공격력 (ATK) | 장비·스킬에 따라 변동 |
| 방어력 (DEF) | 받는 피해 감소 |

### 전투 행동

| 행동 | 내용 |
|------|------|
| 스킬 사용 | 보유 무공 중 선택 |
| 도망 | 40% 확률로 전투 이탈 |

### 전투 후 처리
- 승리 시: 냥(골드) 획득, 무공 숙련도 소량 상승
- 장착 장비 내구도 1~3 감소
- 내구도 0 → 파손 → 자동 해제

## 4. 무공 스킬 시스템

### 핵심 규칙
- **직업 없음** — 원하는 무공을 자유롭게 선택
- **첫 무공** — 시작 시 자유 선택
- **추가 무공 조건** — 보유 무공 중 하나라도 숙련도 50% 이상이면 새 무공 해금 가능
- **스킬 레벨** — 숙련도 100% 달성 시 레벨업 (최대 Lv5)

### 무공 목록

#### 검술 계열
| 무공 | 내공 소모 | 효과 |
|------|---------|------|
| 기본검법 | 0 | 기본 베기 (ATK×1.0) |
| 용아참 | 12 | 강력한 하향 베기 (ATK×1.8) |
| 선풍검 | 18 | 회전 2타 (ATK×1.3 × 2) |
| 반격지세 | 8 | 방어+반격 자세 |

#### 내공 계열
| 무공 | 내공 소모 | 효과 |
|------|---------|------|
| 기공심법 | 15 | 체력 회복 |
| 철포삼 | 10 | 방어 대폭 증가 |
| 기폭장 | 22 | 내공 폭발 공격 (ATK×2.2) |
| 운기조식 | 0 | 내공 회복 |

#### 경공 계열
| 무공 | 내공 소모 | 효과 |
|------|---------|------|
| 환영보 | 8 | 회피 확률 +40% |
| 뇌전신보 | 14 | 번개 질주 공격 (ATK×1.5) |

#### 암기 계열
| 무공 | 내공 소모 | 효과 |
|------|---------|------|
| 비침 | 10 | 독침 + 독 상태 부여 |
| 연막탄 | 8 | 적 명중률 -40% |
| 폭뢰 | 20 | 폭발 공격 (ATK×2.5, 반동 피해 있음) |

## 5. 생활 스킬 시스템

### 채집 수량 시스템
```
모든 생활 스킬 동일 적용
(채집/벌목/채광/낚시/농사)

숙련도 0~19%   → 1개
숙련도 20~39%  → 2개
숙련도 40~59%  → 3개
숙련도 60~79%  → 4개
숙련도 80~100% → 5개

레벨업 시 → 숙련도 0% 리셋 → 1개부터 다시 시작
```

### 플레이 흐름 예시
```
채집 Lv1, 숙련도 0%  → 쑥 1개
채집 Lv1, 숙련도 45% → 쑥 3개
채집 Lv1, 숙련도 82% → 쑥 5개
레벨업! Lv2, 숙련도 0% → 질경이 1개 (리셋)
채집 Lv2, 숙련도 80% → 질경이 5개
```

## 6. 제련/제작 시스템

### 재료량 기준 원칙
```
+10 강화 달성 목표 기간 (하루 8h 직접채집 + 건물 기준):
  Lv1 아이템: 약 3일  → 아이템 1개당 재료 합계 3개
  Lv2 아이템: 약 7일  → 아이템 1개당 재료 합계 8개
  Lv3 아이템: 약 15일 → 아이템 1개당 재료 합계 6개
  Lv4 아이템: 약 30일 → 아이템 1개당 재료 합계 12개
  Lv5 아이템: 약 60일 → 아이템 1개당 재료 합계 25개
```

### 제련 (용광로)

```
철광석×2   → 철괴
강철광석×5 → 강철괴
은광석×4   → 은괴
미스릴원석×8 → 미스릴괴
천강석×15  → 천강괴
```

## 9. 수리 시스템

### 기본 원칙
- **수리 위치**: 가방(인벤토리) 어디서든 즉시 가능
- **수리 비용**: 소실된 내구도 × 1냥
- **최대 내구도 영구 감소**: 수리 1회마다 최초 최대 내구도의 5% 고정 감소
- **감소 한계**: 최초 최대 내구도의 10%까지 감소 (최대 18회 수리 후 하한선 도달)

### 수리 비용 계산

```
수리비 = 최대 내구도 - 현재 내구도 (소실량 × 1냥)
```

예시:
```
철검 (최초 최대 내구도 150)
현재 내구도 80 → 수리 시:
  소실량 = 150 - 80 = 70
  수리비 = 70냥
```

### 최대 내구도 감소 규칙

```
수리 1회당 감소량 = 최초 최대 내구도 × 5% (고정값)
하한선 = 최초 최대 내구도 × 10%

※ 매번 현재 최대 내구도의 5%가 아니라
  최초 최대 내구도 기준 5% 고정 감소
```

예시:
```
철검 최초 최대 내구도: 150
1회당 고정 감소량: 150 × 5% = 7.5 → 올림 → 8
하한선: 150 × 10% = 15

수리 1회: 최대 내구도 150 → 142
수리 2회: 최대 내구도 142 → 134
수리 3회: 최대 내구도 134 → 126
...
수리 17회: 최대 내구도 22 → 15 (하한선 도달)
수리 18회~: 최대 내구도 15 고정 (더 이상 감소 없음)
```

### 수리 불가 조건
- 냥이 부족한 경우
- 현재 내구도 = 최대 내구도 (이미 최대)
- 하한선 도달 후에도 수리는 가능 (최대 내구도 감소만 멈춤)

## 12. 전투 시스템

### 피해 계산
```
기본 피해 = 공격력 - 방어력 (최소 0, 마이너스 없음)
```

### 회피/관통 공식
```
실질 회피 = 회피% - 관통%

특별 규칙:
  회피% = 관통% (단, 둘 다 0 제외) → 50% 고정
  둘 다 0                           → 100% 명중 (기본값)
  결과 0% 이하                      → 0% (무조건 맞음)
  결과 100% 이상                    → 100% (절대 안 맞음)
```

### 크리티컬 시스템
```
크리티컬 피해 = 공격력 × 3.0 (방어력 무시)
크리티컬 시 회피 = 원래 회피율 × 50%
최대 크리티컬 확률 = 100% (극악 조건)
```

### 크리티컬 확률 공식
```
크리티컬 확률 =
  내 운%
  + 장비/스킬/상황 보너스
  + (100 - 상대 운%) × 0.1
  + (100 - 상대 회피%) × 0.05
  + 내 관통% × 0.05
  - 상대 관통% × 0.05
```

### 크리티컬 발생 원인
```
1. 운 스탯          → 운 +1 = 크리티컬 +0.05%
2. 장비 등급        → 土+0.5% / 木+1% / 水+2% / 火+3% / 金+5%
   무기 강화 +10    → 추가 +5%
   악세사리 풀세팅  → 최대 +15%
3. 무공 스킬        → 특정 스킬 사용 시 일시적 +10~20%
4. 상태이상
   독 걸린 적 공격  → +5%
   연속 공격 3회    → +10% 스택
5. 상황 보너스
   체력 30% 이하    → +15% (사생결단)
   뒤에서 공격      → +10%
```

### 몬스터 전투
```
몬스터도 동일한 전투 공식 적용
  → 운/회피/관통 스탯 보유
  → 몬스터 크리티컬도 동일하게 발동
  → 고렙 몬스터일수록 관통력↑ (회피 관통)
  → 몬스터 크리티컬 = 몬스터 공격력 × 3 (방어 무시)
```

## 13. 스킬 통합 시스템

**모든 스킬에 동일하게 적용**
```
무공 스킬  (기본검법, 용아참 등)
생활 스킬  (채집, 벌목, 채광, 낚시, 농사)
제작 스킬  (제련, 제작, 연금 등)
```

### 공통 구조
```
숙련도 0~100%
  → 100% 달성 시 레벨업
  → 최대 Lv10 (현재 구현 Lv5)
  → 숙련도 50% 이상이면 새 스킬 배울 수 있음

오행 등급 적용
  → 스킬 사용 결과물에 등급 부여
  → 등급에 따라 숙련도 +1~5p

레벨에 따라 할 수 있는 것이 달라짐
  → 무공 스킬  : 레벨 높을수록 위력 증가
  → 생활 스킬  : 레벨 높을수록 좋은 재료 획득
  → 제작 스킬  : 레벨 높을수록 고급 아이템 제작 가능
```

### 숙련도 획득 방식
```
무공 스킬  → 전투에서 사용할 때마다 획득
생활 스킬  → 채집/벌목/채광/낚시/농사 할 때마다 획득
제작 스킬  → 제련/제작/연금 할 때마다 획득
```

```
  +1 → +2 : +1 아이템 3개
  ...
  +9 → +10: +9 아이템 3개
```

### 보스 몬스터
```

예) 메뚜기 50마리 → 보스메뚜기 출현

보스 드롭 = 희귀 재료 추가
```

### 보스 카운터 시스템
```
사냥터 UI 상단에 항상 표시
  예) 메뚜기 처치: 43/50

카운터 유지 조건:
  → 보스 출현 전 사냥터 나가면 카운터 그대로 유지
  → 10마리 잡고 나가면 다음 입장 시 10마리부터 시작

카운터 초기화 조건:
  → 보스 출현 후 "나중에" 선택 시 초기화 (0/50)
  → 보스 처치 시 초기화 (0/50)
```

```

  → 선택지:
```

```
  → 일반 몬스터 리젠 없음

  → 나가면 카운터 초기화
```

```
운 스탯이 높을수록 확률 상승
추후 문제 발생 시 수치 조정
```

```
풀어쓰면:

소수점 올림 처리
```

## 31. 방패 시스템

### 방패 부품 구성
```
방패 = 방패판 + 손잡이 + 문양석

방패판  → 무기제작 스킬 (금속)  → 방어력 담당
손잡이  → 목공 스킬 (나무/가죽) → 체력 보너스 담당
문양석  → 악세사리 스킬 (보석)  → 반격 확률 담당
```

### 시작 단계 — 나무/가죽 아이템
```
제작 Lv0 (시작): 나무 무기 + 가죽 방어구
제작 Lv1       : 철 무기/방패 + 기본 철 방어구
제작 Lv2       : 강철 무기/방패 + 강철 방어구
제작 Lv3       : 은 무기/방패 + 은 방어구
제작 Lv4       : 미스릴 무기/방패 + 미스릴 방어구
제작 Lv5       : 천강 무기/방패 + 천강 방어구
```

### 나무 무기 재료
```
[나무 검]
칼날  : 나무칼날   (목재×2 + 나뭇가지×1)
칼자루: 나무자루   (나뭇가지×2 + 풀잎×1)
칼집  : 풀칼집    (풀잎×3 + 나무껍질×1)
→ 조합 → 나무검

[나무 방패]
방패판 : 나무방패판  (목재×3 + 나뭇가지×1)
손잡이 : 기초 손잡이 (나뭇가지×2 + 풀잎×1)
문양석 : 없음
→ 조합 → 나무방패
```

### 가죽 방어구 재료 (시작)
```
머리 : 풀잎모자  (풀잎×5)
몸통 : 가죽조끼  (나무껍질×3 + 풀잎×2)
팔   : 가죽팔찌  (나무껍질×2)

발   : 가죽신발  (나무껍질×2 + 수지×1)
```

```
  레벨업마다 +1칸 확장 가능 (냥 + 재료)
  최대 100칸

[소모품]
  100개 중첩
  장비 칸에 포함
```

### 채집 → 창고 흐름
```
채집 → 가방 (칸 사용)
가방 꽉 차면 → 마을 창고로 이동
창고 재료 → 칸 개념 없이 수량만 저장
```

```

Lv2  → 채집 Lv2 재료

Lv10 → 채집 Lv10 재료
```

```
  Lv4 업그레이드: 냥 + 은광석 20개
  ...
  (수치는 플레이 후 조정)
```

```

3. 사냥터 밖 시간 경과   → 자동 회복
4. 음식 섭취              → 추후 구현
```

### 내공 회복
```
1. 내공단 아이템    → 즉시 회복

3. 칼집 장착 시    → 자동 회복 (전투 중/외)
4. 사냥터 밖 시간  → 자동 회복
```

### 공통 원칙


## 26. 몰이 사냥 시스템

### 기본 구조
```
기본: 1마리씩 사냥
전체 누적 처치 500마리마다 동시 사냥 +1마리

0~499마리    → 1마리 동시
500~999마리  → 2마리 동시
1000~1499마리→ 3마리 동시
1500~1999마리→ 4마리 동시
2000마리~    → 5마리 동시
```

### 카운터 기준
```
최대레벨 몬스터 기준으로 카운터 산정
저렙 몬스터도 동일하게 카운트
최대레벨 확장 시 자연스럽게 적용
```

### 내구도 소모
```
내구도 소모 = MAX(스킬 사용 횟수, 동시 사냥 마리수)

스킬 사용 시 무조건 1 소모

예시 1) 2마리 동시, 스킬 5번 사용
  → MAX(5, 2) = 5 소모

예시 2) 3마리 동시, 스킬 2번에 전멸
  → MAX(2, 3) = 3 소모

예시 3) 5마리 동시, 스킬 1번에 전멸
  → MAX(1, 5) = 5 소모
```

### 몰이 사냥 보상
```
동시 사냥 마리수만큼 경험치/드롭 배수 적용
2마리 동시 → 경험치/드롭 ×2
3마리 동시 → 경험치/드롭 ×3
```

## 52. 퀘스트 시스템

```
1. 메인 퀘스트 (스토리)
```

```
3. 일일 퀘스트
```

### 퀘스트 보상
```
재료 + 특수아이템
(냥/경험치는 사냥/채집으로 획득)
→ 퀘스트만의 고유 보상!
```

### 메인 퀘스트

```
[1장] 프롤로그 — 주막
  조건: 시작
  보상: 목검 + 나무방패 (지급품)

[2장] 마을 어귀
  조건: Lv3 달성
  보상: 쑥×10 + 풀잎×20

[3장] 대장장이 오씨
  조건: 제련 스킬 Lv2 달성
  보상: 철괴×3 + 부싯돌×5

[4장] 첫 번째 철검
  조건: 철검 1개 제작 완료
  보상: 강철괴×2 + 구리광석×5

[5장] 뒷산 도전
  조건: Lv10 달성
  보상: 미스릴원석×1 + 자수정×2

[6장] 청룡방 퇴치
  조건: 혈귀 장옥 처치
  보상: 천마신공 두루마리 (특수아이템)
       + 천강석×1
```

### 일반 퀘스트 (NPC별)

#### 대장장이 오씨
```
의뢰1: 철광석 10개 가져다줘
  보상: 철괴×3

의뢰2: 강철검 1개 제작해줘
  보상: 미스릴원석×1

의뢰3: 철갑옷 풀셋 제작
  보상: 미스릴괴×2 + 자수정×3
```

#### 약초꾼 이씨
```
의뢰1: 쑥 20개 채집해줘
  보상: 회복단×3

의뢰2: 도라지 10개 채집해줘
  보상: 대회복단×2 + 인삼×1

의뢰3: 인삼 5개 채집해줘
  보상: 해독제×2 + 산삼×1
```

#### 원로 방씨
```
의뢰1: 산토끼 5마리 처치
  보상: 토끼가죽×5 + 토끼뼈×3

의뢰2: 메뚜기 보스 처치
  보상: 강화메뚜기껍질×3 + 수정×2

의뢰3: 청룡방 졸개 10명 처치
  보상: 천마신공 파편 (특수아이템)
       + 마교검파편×5
```

### 특수아이템 종류
```
천마신공 두루마리 → 특정 무공 스킬 해금
천마신공 파편    → 수집 시 특수 스킬 해금
희귀 제작 레시피 → 특수 아이템 제작 가능
칭호            → 캐릭터 이름 옆에 표시 (추후)
```

### 추후 추가 예정
```
일일 퀘스트  → 매일 초기화, 재료 보상
업적 퀘스트  → 누적 달성, 칭호/특수 보상
퀘스트 수    → 현재 6개(메인) + 9개(일반) = 15개
              추후 계속 추가
```

### 해금 조건
```
캐릭터 Lv10 달성 시 자동 사냥 해금
```

### 경험치/드롭
```
자동 사냥 경험치: 직접 사냥의 20%
자동 사냥 드롭:   직접 사냥의 20%
```

### 자동 사냥 설정 (유저가 직접 설정)

#### 기본 설정
```
사냥터 선택
몰이 마리수 설정
```

#### 체력 관리
```
조건: __% 이하
행동:
  ○ 회복단 자동 사용
  ○ 기공심법 스킬 자동 사용
  ○ 회복단 + 기공심법 둘 다 사용
  ○ 사냥 중단
  ○ 무시하고 계속

회복 우선순위:
  ○ 스킬 먼저 → 없으면 아이템
  ○ 아이템 먼저 → 없으면 스킬
```

#### 내공 관리
```
내공 소진 시:
  ○ 운기조식 스킬 자동 사용
  ○ 내공단 아이템 자동 사용
  ○ 운기조식 + 내공단 둘 다 사용
  ○ 기본 공격만 계속
  ○ 사냥 중단

회복 우선순위:
  ○ 스킬 먼저 → 없으면 아이템
  ○ 아이템 먼저 → 없으면 스킬
  ○ 스킬만 사용
  ○ 아이템만 사용
```

#### 내구도 관리
```
교체 조건:
  ○ 내구도 0% (파손) 시 교체
  ○ 내구도 10% 이하 시 교체
  ○ 내구도 20% 이하 시 교체
  ○ 내구도 30% 이하 시 교체
  ○ 내구도 50% 이하 시 교체

교체 우선순위 (가방 내 예비 장비):
  ○ 등급 높은 것 먼저
  ○ 강화 높은 것 먼저
  ○ 내구도 높은 것 먼저
  ○ 가방 순서대로

교체 불가 시 (예비 장비 없을 때):
  ○ 사냥 중단
  ○ 파손까지 계속 사냥
```

#### 가방 관리
```
가방 가득 찰 시:
  ○ 사냥 중단
  ○ 재료 버리고 계속
  ○ 장비 버리고 계속
```

### 부하 여부
```
자동 사냥은 클라이언트(로컬 PC)에서 처리
설정값은 SQLite에 저장 (용량 거의 없음)
```
